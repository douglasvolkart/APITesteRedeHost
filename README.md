############# Instruções para rodar os projetos ############

 - Rodar o projeto API no visual studio ou no servidor IIS e anotar o dominio e porta.

 - Configurar o arquivo app.js a variavel da api que faz parte do projeto de front end, na pasta assets/js/ alterando o dominio de testes "localhost:58193" pelo dominio e porta de testes utilizados na api.

- Rodar o StaticServer node server.js
